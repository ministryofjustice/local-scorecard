Accessibility statement for Criminal Justice System Delivery Data Dashboard

Introduction

This accessibility statement applies to the March 2022 publication of the Criminal Justice System (CJS) Delivery Data Dashboard website criminal-justice-delivery-data-dashboards.justice.gov.uk

This website is run by the Ministry of Justice. We want as many people as possible to be able to use this website.

We’ve also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.


How accessible this website is

We know that some parts of this website are not fully accessible:

  • data visualisation tools in the website currently use inaccessible technologies such as Tableau
  • we have a small number of attached documents which are in PDF format, and haven't been designed for accessibility

Reporting accessibility problems with this website

We’re always looking to improve the accessibility of this website. if you find any problems not listed on this page or think we’re not meeting accessibility requirements, contact:

gemma.styles@justice.gov.uk


Reporting accessibility problems with this website

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the accessibility regulations. If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).


Technical information about this website’s accessibility

The Ministry of Justice is committed to making its website accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

This website is partially compliant with the Web Content Accessibility Guidelines version 2.1 AA standard, due to ‘the non-compliances’ listed below.


Non-accessible content

The content listed below is non-accessible for the following reasons.


Non-compliance with the accessibility regulations

Interactive tools

We are aware that we have Tableau visualisations on the website which is not an accessible software item, and cannot be made accessible. In some cases the information is not easily available to users in an accessible format. We are currently investigating more accessible visualisation tools. If you require information from these tools, then please contact us.

Contrast and use of colour

Some colours are not accessible or lack contrast with the background, for example the maps.

We are working to improve these problems and will look to use a more accessible colour palette in future iterations.

Keyboard navigation

If you are navigating with keyboard only, you cannot tab to some parts of the content. This affects graph-viewing tools that appear on hover, such as the zoom button and tooltips.

The graphs are created using third party software, so unfortunately we cannot fix this issue.

Images

Some complex images (graphs and interactive maps) do not have a suitable text alternative. The images are created using third party software and we are in the process of investigating more accessible visualisation tools. All data contained in the graphs and maps are available to download in alternative formats, including CSV.

Alternative formats

The introduction and metric documentation tabs are static HTML pages, making them easier to navigate. Some information is also available to download as a Data Note.


Preparation of this accessibility statement

This statement was prepared on 25 March 2022. It was last reviewed on 25 March 2022.

This website was last tested on 25 March 2022. The tests were based on a design review and carried out internally.
